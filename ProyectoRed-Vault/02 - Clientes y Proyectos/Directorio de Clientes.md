# 👥 Directorio de Clientes

## Cómo usar esta nota

Para agregar un cliente, copia la plantilla de abajo y llena los datos. Cada cliente importante puede tener su propia nota vinculada.

---

## Plantilla de cliente

```
## [Nombre de la empresa]

| Campo | Detalle |
|---|---|
| **Giro** | (Industrial / Restaurante / Edificio / Bomberos / Otro) |
| **Contacto** | Nombre del responsable |
| **Teléfono** | |
| **Correo** | |
| **Dirección** | |
| **Estado** | CDMX / EdoMex / Querétaro / Puebla / Aguascalientes / BC |
| **Cliente desde** | YYYY-MM-DD |
| **Estatus** | Activo / Prospecto / Inactivo |

**Productos/servicios contratados:**
-

**Notas:**
-
```
